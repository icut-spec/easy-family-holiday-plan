# US-2.4 — Holiday type selector

**As a user**, I want to choose a holiday type (Beach, Mountains, City, Camping, Other), so that the app can tailor activity and packing suggestions to my trip.

## Acceptance criteria

- Selector with options: Beach, Mountains, City, Camping, Other
- Selected type is saved to state
- Packing list seeds and activity suggestions update based on the selected type
- Default selection is "Other" if none chosen

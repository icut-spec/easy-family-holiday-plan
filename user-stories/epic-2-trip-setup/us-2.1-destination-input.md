# US-2.1 — Destination input

**As a user**, I want to type in my holiday destination, so that the app can use it to generate activity suggestions and map links.

## Acceptance criteria

- Text input field for destination (e.g. "Paris", "Barcelona")
- Destination is saved to app state on change
- Destination value is used by the Map and Activities sections
- Field is required before the trip can be saved

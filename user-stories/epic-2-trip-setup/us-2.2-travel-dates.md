# US-2.2 — Travel dates

**As a user**, I want to enter a start and end date, so that I can see the duration of my trip and plan accordingly.

## Acceptance criteria

- Start date and end date fields (date pickers or text inputs)
- End date must be on or after start date; validation error shown otherwise
- Trip duration (number of days) is calculated and displayed
- Dates are used to scaffold the day-by-day itinerary (Epic 9)

# US-3.3 — Add/remove pets

**As a user**, I want to add pets with their name and type (e.g. dog, cat), so that the app can filter pet-friendly activities and add pet-related packing items.

## Acceptance criteria

- Input fields for pet name and type (e.g. dog, cat, bird)
- Each pet appears in a list with a remove button
- When one or more pets are present, pet-specific packing items are added automatically
- Pet count and type are reflected in the trip summary and trip card

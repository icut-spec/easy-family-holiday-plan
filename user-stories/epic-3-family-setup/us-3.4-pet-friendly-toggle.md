# US-3.4 — Pet-friendly toggle

**As a user**, I want a global pet-friendly toggle, so that I can quickly filter all activities to only show ones that allow pets.

## Acceptance criteria

- Toggle switch labelled "Pet-friendly only" or similar
- When enabled, only pet-friendly activities are shown in the Activities tab
- Toggle state is saved to state and persists on refresh
- Toggle is automatically enabled when at least one pet is added

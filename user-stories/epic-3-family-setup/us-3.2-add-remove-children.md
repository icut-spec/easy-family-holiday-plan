# US-3.2 — Add/remove children

**As a user**, I want to add children with their name and age, so that the app can tailor activities and packing items for kids.

## Acceptance criteria

- Input fields for child name and age
- Each child appears in a list with a remove button
- When one or more children are present, kid-specific packing items are added automatically
- Child count is reflected in the trip summary

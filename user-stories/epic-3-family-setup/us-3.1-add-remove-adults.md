# US-3.1 — Add/remove adults

**As a user**, I want to add and remove adult family members by name, so that I know who is on the trip.

## Acceptance criteria

- Input field to add an adult by name
- Each added adult appears in a list with a remove button
- At least 1 adult is required; removing the last adult is prevented
- Adult count is reflected in the trip summary and trip card

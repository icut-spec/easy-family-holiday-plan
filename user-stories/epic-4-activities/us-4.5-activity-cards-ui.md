# US-4.5 — Activity cards UI

**As a user**, I want each activity displayed as a card with a name, short description, pet-friendly badge, and Maps link, so that the information is easy to scan.

## Acceptance criteria

- Each card shows: activity name, short description, pet-friendly badge (if applicable), "Open in Maps" link
- Pet-friendly badge is visually distinct (e.g. a paw icon or green label)
- Cards are responsive and readable on mobile and desktop
- Empty state shown if no activities match the current filters

# US-6.1 — Mobile responsive layout

**As a user on a phone**, I want the app to work well on small screens, so that I can plan on the go.

## Acceptance criteria

- All pages and components are usable on screens 375px wide and above
- Navigation tabs are accessible and tappable on mobile
- No horizontal scrolling on any screen
- Text, buttons, and inputs are appropriately sized for touch interaction
- Tested at common breakpoints: 375px, 768px, 1280px

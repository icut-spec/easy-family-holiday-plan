# US-6.3 — Trip summary view

**As a user**, I want a quick summary at the top showing destination, dates, number of travelers, and budget, so that I can see the key details at a glance.

## Acceptance criteria

- Summary is visible on the Trip Overview page
- Displays: destination, start/end dates, trip duration, number of adults/children/pets, total budget
- Updates immediately when any of these values are changed
- Summary is compact and does not dominate the screen

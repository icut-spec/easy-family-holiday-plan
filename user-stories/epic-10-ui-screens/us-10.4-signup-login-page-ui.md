# US-10.4 — Sign-up / Login Page UI

**As a new or returning user**, I want a clean, simple authentication page, so that signing up or logging in feels fast and trustworthy.

## Acceptance criteria

- Tabbed or toggled layout with "Sign up" and "Log in" on the same page
- Sign-up fields: name, email, password
- Login fields: email, password
- "Continue with Google" OAuth button present on both tabs
- Inline validation errors shown (e.g. "Email already in use", "Password too short")
- Link back to the landing page
- Responsive: works well on mobile (375px+)

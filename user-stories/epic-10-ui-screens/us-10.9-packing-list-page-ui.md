# US-10.9 — Packing List Page UI

**As a user**, I want a well-organised packing list page with clear category sections, so that packing is fast and stress-free.

## Acceptance criteria

- Items grouped under category headings: Family, Pets, Documents, Essentials, Kids, Activities, holiday-type categories
- Each item has a checkbox, label, and remove button
- Checked items are visually distinct (strikethrough or moved to bottom of section)
- "Add item" form at the bottom of the list
- Progress bar at the top: "X / Y packed"

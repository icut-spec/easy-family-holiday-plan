# US-10.5 — Dashboard: My Trips screen

**As a logged-in user**, I want a dashboard that shows all my trips at a glance, so that I can quickly navigate to the one I'm planning.

## Acceptance criteria

- Page heading: "My Trips"
- Each trip shown as a card: trip name, destination, date range, holiday type, traveller summary (e.g. "2 adults · 1 child · 1 dog")
- "Create Trip" button is prominent at the top
- Empty state: "No trips yet. Start planning your first trip!"
- Trip cards are clickable and navigate to the Trip Overview page

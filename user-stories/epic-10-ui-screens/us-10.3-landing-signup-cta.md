# US-10.3 — Landing Page: Sign-up CTA section

**As a visitor**, I want a clear sign-up prompt at the bottom of the landing page, so that I can create an account without scrolling back to the top.

## Acceptance criteria

- Dedicated CTA section near the footer with a headline and "Sign up free" button
- Button links to the sign-up page
- Section is visually distinct from the rest of the page (e.g. coloured background)

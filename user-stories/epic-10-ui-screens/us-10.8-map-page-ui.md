# US-10.8 — Map Page UI

**As a user**, I want a full-screen map page centred on my destination, so that I can visually explore pet-friendly places.

## Acceptance criteria

- Full-width embedded Google Map centred on `trip.destination`
- Coloured pins: parks (green), pet-friendly restaurants (orange), veterinarians (red)
- Sidebar or bottom sheet listing places shown on the map
- Clicking a pin shows a popup: name, category, address, "Get Directions" link
- On mobile the sidebar becomes a collapsible bottom sheet

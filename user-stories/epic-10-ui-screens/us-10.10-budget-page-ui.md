# US-10.10 — Budget Page UI

**As a user**, I want a simple, readable budget page that shows my spending by category and a running total, so that I can stay on track financially.

## Acceptance criteria

- Editable budget limit field at the top
- Expense rows show: category, description, and amount (e.g. "Hotel · €400")
- Running total displayed at the bottom: "Total · €700"
- Visual indicator (red highlight or warning) if total exceeds budget limit
- "Add expense" form: category (select), description, amount
- Category options: Accommodation, Food, Transport, Activities, Other

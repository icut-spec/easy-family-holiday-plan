# US-10.2 — Landing Page: Features section

**As a visitor**, I want to see a clear explanation of what the app does and its key features, so that I can decide if it's right for my family.

## Acceptance criteria

- "How it works" section shows a 3-step explainer: Create a trip → Add your family & pets → Plan everything
- Features section shows at least 4 feature cards, each with an icon, title, and short description:
  - Day-by-day itinerary
  - Pet-friendly map
  - Smart packing list
  - Budget tracker
- Features section is responsive: 2-column grid on desktop, 1 column on mobile

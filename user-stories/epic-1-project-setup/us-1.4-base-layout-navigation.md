# US-1.4 — Base layout & navigation

**As a user**, I want a clean single-page layout with tab navigation between the 4 sections, so that I can move between them easily.

## Acceptance criteria

- `index.html` is the single entry point
- Tab navigation links to: Trip Setup, Family, Activities, Packing List
- Active tab is visually highlighted
- Layout is readable on both mobile and desktop

# US-5.5 — Categorized packing items

**As a user**, I want packing items grouped by category (Documents, Clothing, Kids, Pets, Activities, etc.), so that packing is easier to organize.

## Acceptance criteria

- Items are displayed under clearly labelled category headings
- Categories include: Family, Documents, Essentials, Kids, Pets, Activities, and holiday-type-specific categories
- Category order follows a defined priority (`CATEGORY_ORDER`)
- Custom items are added to a "Custom" category or the user can choose a category
- Categories with no items are hidden

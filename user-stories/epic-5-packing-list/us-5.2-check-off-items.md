# US-5.2 — Check off items

**As a user**, I want to check off packing items as I pack them, so that I can track my progress.

## Acceptance criteria

- Each item has a checkbox
- Checking an item marks it as packed (visually distinct, e.g. strikethrough)
- Checked state is saved to state and persists on refresh
- A progress counter is shown (e.g. "12 / 24 packed")

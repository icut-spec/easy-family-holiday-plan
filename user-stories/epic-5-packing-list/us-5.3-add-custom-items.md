# US-5.3 — Add custom items

**As a user**, I want to add my own custom packing items to the list, so that I can include things the app didn't suggest.

## Acceptance criteria

- An "Add item" input field is available at the bottom of the packing list
- User can type a custom item name and press Enter or click Add
- Custom items appear in the list and behave identically to seeded items (checkable, removable)
- Empty input is rejected with no action

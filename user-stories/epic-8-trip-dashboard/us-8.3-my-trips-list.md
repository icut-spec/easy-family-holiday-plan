# US-8.3 — My Trips list

**As a user**, I want to see a list of all my saved trips, so that I can switch between them or start planning a new one.

## Acceptance criteria

- Trips list shows each trip as a card with: name, destination, date range, holiday type, family summary (e.g. "2 adults · 1 child · 1 dog")
- "New Trip" / "Create Trip" button is prominent at the top of the list
- Clicking a trip card opens its dashboard
- Empty state shown when no trips exist: "No trips yet. Create your first trip!"
- List updates immediately when a trip is added or deleted

# US-8.2 — Trip Dashboard layout

**As a user**, after creating a trip I want to see a dashboard with four sections — Itinerary, Map, Packing List, and Budget — so that I can plan every aspect of the trip from one place.

## Acceptance criteria

- Dashboard is the landing view immediately after trip creation or when opening an existing trip
- Four clearly labelled tabs: Overview, Itinerary, Map, Packing List, Budget
- Trip name and destination are shown prominently at the top of the dashboard
- A "Back" button or "My Trips" link returns the user to the trips list
- Active tab is visually highlighted

# Epic 3: Family Setup

> Allow the user to define who is coming on the holiday.

---

## US-3.1 — Add/remove adults

**As a user**, I want to add and remove adult family members by name, so that I know who is on the trip.

**GitHub Issue:** #11

---

## US-3.2 — Add/remove children

**As a user**, I want to add children with their name and age, so that the app can tailor activities and packing items for kids.

**GitHub Issue:** #12

---

## US-3.3 — Add/remove pets

**As a user**, I want to add pets with their name and type (e.g. dog, cat), so that the app can filter pet-friendly activities and add pet-related packing items.

**GitHub Issue:** #13

---

## US-3.4 — Pet-friendly toggle

**As a user**, I want a global pet-friendly toggle, so that I can quickly filter all activities to only show ones that allow pets.

**GitHub Issue:** #14

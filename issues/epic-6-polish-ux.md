# Epic 6: Polish & UX

> Ensure the app is responsive, accessible, and pleasant to use.

---

## US-6.1 — Mobile responsive layout

**As a user on a phone**, I want the app to work well on small screens, so that I can plan on the go.

**GitHub Issue:** #25

---

## US-6.2 — Empty state messages

**As a user**, I want helpful prompts when a section is empty (e.g. "Enter a destination to see activities"), so that I always know what to do next.

**GitHub Issue:** #26

---

## US-6.3 — Trip summary view

**As a user**, I want a quick summary at the top showing destination, dates, number of travelers, and budget, so that I can see the key details at a glance.

**GitHub Issue:** #27

---

## US-6.4 — Reset / start new plan

**As a user**, I want a "Start New Plan" button that clears all data, so that I can plan a new trip from scratch.

**GitHub Issue:** #28
